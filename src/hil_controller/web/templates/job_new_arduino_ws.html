Execution location

Each job phase can run on the controller or the DUT host. rpi-displays is too weak to compile WipperSnapper, so build defaults to the controller; flashing reaches the DUT over usbip (the controller attaches the DUT's USB port) or by shipping the built artifacts to the DUT host.

pio run (compile) host

Must match the build host (shared workspace)
